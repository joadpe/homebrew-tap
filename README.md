# Homebrew Tap

Custom Homebrew formulae by [joadpe](https://github.com/joadpe).

## Install

```bash
brew tap joadpe/tap
brew install jsrc
```

## Formulae

| Formula | Description |
|---------|-------------|
| [jsrc](https://github.com/joadpe/jsrc) | Java source code navigator and analyzer |

## Usage

```bash
jsrc --help
cd /path/to/java/project
jsrc --index
jsrc --overview --json
jsrc --classes --json
jsrc --smells --json
```
